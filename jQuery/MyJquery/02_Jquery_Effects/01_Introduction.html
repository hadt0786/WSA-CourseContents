J query has the following effects
  1. hide
  2. Show
  3. Toggle
  4. Slide
  5. Fade
  6. Animate

Hide and Show

syntax
$(selector).hide(speed, callback); $(selector).hide(speed, callback);
Example $("button").click(function(){ $(".p1").show(); $(".p2").hide(); });

Hide and Show with Speed parameters

$("button").click(function(){ $(".p1").show(1000); $(".p2").hide("slow"); $(".p3").hide("fast"); });

Hide and Show with Calback

$("button").click(function(){ $(".p1").hide(1000, function(){ alert("hide() method is finished"); }); $(".p2").show(1000, function(){ alert("show() method is finished"); }); });

Toggle

Syntax: $(selector).toggle(speed, callback); Example : $("button").click(function(){ $(".p1").toggle(); });

Toggle with Speed parameters

$("button").click(function(){ $(".p1").toggle(2000); $(".p2").toggle("slow"); $(".p3").toggle("fast"); });

Toggle with Calback

$("button").click(function(){ $(".p1").toggle(1000, function(){ alert("toggle() method is finished"); }); });
